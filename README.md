# SNEStoN64
Este ejecutable facilita la conversión de ROMs de SNES a .z64

### Controles
|  **N64**  |       **SNES**        |
|:---------:|:---------------------:|
| C-Buttons |         ABXY          |
|   D-Pad   |         D-Pad         |
|    L/R    |          L/R          |
|    A/B    |     Start/Select      |
|   Start   |  Toggle FPS Counter   |
|     Z     | Toggle Layer Priority |
